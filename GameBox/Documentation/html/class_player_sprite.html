Layer *  caller 
) [virtual]

Initializes a sprite with an image filename.

This method will find pszFilename from local file system, load its content to Texture2D, then use Texture2D to create a sprite. After initialization, the rect used will be the size of the image. The offset will be (0,0).

Parameters:
pszFilenameThe path to an image file in local file system
Returns:
true if the sprite is initialized properly, false otherwise.
